# KodNest Careers

Monorepo for the KodNest Careers platform.

## Structure
- apps/        → Microservices & frontend
- libs/        → Shared domain, contracts, utilities
- infra/       → Docker, Kubernetes, Terraform
- docs/        → Architecture & decisions
- scripts/     → DB & maintenance scripts
- tests/       → E2E, integration, performance tests

## Tech Stack
- Backend: Node.js (NestJS)
- Frontend: Next.js
- Database: PostgreSQL
- Messaging: Kafka / RabbitMQ
- Infra: Docker, Kubernetes
